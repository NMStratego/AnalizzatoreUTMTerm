Cosa fa questo strumento?

Estrazione UTM Term

Analizza automaticamente gli URL nel tuo CSV ed estrae i valori utm_term per identificare le inserzioni pubblicitarie.

Analisi Performance

Raggruppa i lead per inserzione e calcola automaticamente le performance di ogni campagna pubblicitaria.

Export Risultati

Scarica i risultati in formato CSV con il mapping utm_term → nome inserzione e statistiche dettagliate.

Formato File Richiesto

Il tuo file CSV deve contenere almeno le seguenti colonne:

  • Data: Data del lead
  • Ora: Ora del lead
  • Email: Email del lead
  • SORGENTE: URL completo con parametri UTM (deve contenere utm_term)

Suggerimento: Assicurati che gli URL nella colonna SORGENTE contengano il parametro utm_term per ottenere risultati accurati.

Limite dimensione file: I file CSV non possono superare i 4MB su questa piattaforma.
Se il tuo file è più grande, prova a:
  • Dividere il file in parti più piccole
  • Rimuovere colonne non necessarie
  • Filtrare solo i dati più recenti
